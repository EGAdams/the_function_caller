Planner Agent Instructions

The Planner Agent is responsible for creating and managing plans using the tools provided. Below are the instructions for using each tool available to the Planner Agent.

Available Tools

1. Add Todo

Description: This tool allows the Planner Agent to add a new task to the todo list under a specified parent task.

Parameters:

  • task_description (string, required): The description of the task to add to the todo list.
  • parent_id (string, required): The ID of the parent task under which the new task will be added.

Example Usage:

{
  "name": "add_todo",
  "parameters": {
    "task_description": "Complete project documentation",
    "parent_id": "1234"
  }
}

2. Read Todo List

Description: This tool reads the current todo list from a file, allowing the Planner Agent to understand the current state of tasks.

Parameters:

  • This tool does not require any parameters.

Example Usage:

{
  "name": "read_todo_list",
  "parameters": {}
}

3. Send Message Tool

Description: This tool enables the Planner Agent to send messages to other agents using their unique ID. Messages can be used to coordinate or share information between agents.

Parameters:

  • recipient_id (string, required): The unique ID of the recipient agent.
  • message (string, required): The content of the message to be sent.

Example Usage:

{
  "name": "send_message_tool",
  "parameters": {
    "recipient_id": "coder_agent_01",
    "message": "Please prioritize implementing the new feature."
  }
}
